# Growth Infrastructure Docs

All documents in this directory are **planning templates**, not published forecasts or commitments.

Financial figures (MRR tables, P&L projections, pricing tiers, ARR targets), named prospects, partner lists, and percentage breakdowns are **placeholder values** generated to scaffold the template structure — **not** Justice OS's actual numbers, plans, or contractual commitments.

**Before any external use** (investor meetings, board decks, partner outreach, grant applications):

1. Replace every `[bracketed]` field.
2. Replace every numeric figure with verified, sourced numbers.
3. Replace prospect/partner lists with real, vetted targets.
4. Have the relevant document reviewed by counsel if it will be relied on by investors, customers, or grantors.

Leaving placeholder values in place and sending these externally creates material misrepresentation risk — for investor-facing documents especially, this is fiduciary-adjacent.
